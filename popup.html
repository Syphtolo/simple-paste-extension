Welcome to the Simple Paste Extension!

To customize the extension, go to this link!

For help, go to the extension page.

If you want the old popup, press Control + Shift + 3, though switching over to the new one by adding the accents through the customization tab is recommended.

The Simple Paste Extension now allows global access! This means that you can use the UI from any application on your computer, from Chrome to Zoom to even a code editor! If you want to open the main extension through chrome, still press Control + M. If you want to open the spanish only UI, press Control + Shift + 3 (the spanish UI is global). If you want to open the main UI globally, press Control + Shift + 1!
